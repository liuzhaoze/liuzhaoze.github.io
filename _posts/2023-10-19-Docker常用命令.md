## 启动容器

```bash
docker run -it --name <容器名称> <镜像名>:<Tag> /bin/bash
```

示例：

```bash
# 启动一个 Ubuntu 22.04 的容器，命名为 test
docker run -it --name test ubuntu:22.04 /bin/bash
```

* 如果要后台运行，使用选项 `-itd`
