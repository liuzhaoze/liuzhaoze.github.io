## 光标移动

|快捷键|功能|
|:-:|:-:|
|<kbd>Ctrl</kbd> + <kbd>A</kbd>|移动到行首|
|<kbd>Ctrl</kbd> + <kbd>E</kbd>|移动到行尾|
|<kbd>Alt</kbd> + <kbd>Left Arrow</kbd>|向前移动一个单词|
|<kbd>Alt</kbd> + <kbd>Right Arrow</kbd>|向后移动一个单词|
|<kbd>Ctrl</kbd> + <kbd>B</kbd>|向前移动一个字符|
|<kbd>Ctrl</kbd> + <kbd>F</kbd>|向后移动一个字符|

## 编辑文本

|快捷键|功能|
|:-:|:-:|
|<kbd>Ctrl</kbd> + <kbd>U</kbd>|剪切光标所在行的内容|
|<kbd>Ctrl</kbd> + <kbd>K</kbd>|剪切从光标位置到行尾的内容|
|<kbd>Ctrl</kbd> + <kbd>W</kbd>|剪切光标前的一个单词|
|<kbd>Ctrl</kbd> + <kbd>Y</kbd>|粘贴最近剪切的内容|
|<kbd>Ctrl</kbd> + <kbd>D</kbd>|删除光标所在位置的字符|
|<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>-</kbd>|撤销上一步操作|

## 功能相关

|快捷键|功能|
|:-:|:-:|
|<kbd>Ctrl</kbd> + <kbd>C</kbd>|终止当前命令|
|<kbd>Ctrl</kbd> + <kbd>L</kbd>|清屏|
|<kbd>Ctrl</kbd> + <kbd>R</kbd>|搜索历史命令|
